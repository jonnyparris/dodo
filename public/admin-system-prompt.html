Global System Prompt

A preamble prepended to the system prompt for every session across every user. Use this for organisation-wide rules, conventions, or context that should never be skipped.

This prefix layers above each user's personal systemPromptPrefix, which in turn layers above Dodo's hardcoded base prompt. Capped at 4 KB. Active sessions pick up changes within ~60 seconds (TTL cache).

How it works

Saved values are stored on the SharedIndex DO under global_config. The CodingAgent DO fetches the value at the start of every prompt turn (with a 60-second TTL cache) and assembles the final system prompt as admin-prefix → user-prefix → base prompt → skills → workspace. To clear the prefix, save an empty value or hit Clear.
